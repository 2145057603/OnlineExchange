# 卡面索引工坊

纯静态 PCK 解析页面。所有 PCK 读取、SHA-256 计算、JSON 提取和索引生成均在浏览器本地完成；页面不使用第三方依赖、分析服务或网络请求。

## Cloudflare Pages 部署

若仓库根目录包含本目录：

1. 在 Cloudflare Pages 连接仓库。
2. 构建命令留空。
3. 构建输出目录填写 `card-art-indexer`。
4. 部署。

若将本目录作为独立仓库根目录，构建输出目录填写 `/`。

`_headers` 已设置 CSP，禁止页面发起连接请求。

## 本地索引

输出 JSON 记录 PCK 文件名、大小、SHA-256、可选模组 ID/版本，以及已识别的 `cardId -> resourcePath` 映射。游戏模组导入时应再次校验本地 PCK 哈希和资源路径。

## 限制

- 当前支持目录中未标记压缩的 Godot PCK 条目。
- 动态生成卡图、没有卡牌标识且无法通过路径推断的资源无法自动映射。
- 网站只生成索引，不上传或分发 PCK、图片与其他模组资源。
